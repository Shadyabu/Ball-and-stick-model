This code works through modelling diffusion inside of a brain. 

In order to run it download the following data:
https://www.dropbox.com/scl/fi/nennx4srzhoywri4dega7/data_p1.zip?e=2&file_subpath=%2Fdata.mat&rlkey=ldqygvrdg81phe07tscakrm0h&dl=0

Add the data.mat file into directory and you can run Question1, Question2 and Question3 files in the Matlab Terminal.

Question1 explores fitting a Ball and Stick Model to estimate diffusion in the brain.
Question2 explores uncertainty quantification in the model.
Question3 explores model selection.
